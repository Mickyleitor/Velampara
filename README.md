# Velampara
This code let the user switch off blowing a microphone and switch on with a lighter on a thermistor a generic lamp

# We need:
* Pre-drilled plate
* Resistors
* Sockets
* Relay 5V
* Bipolar Transistor NPN 2N3904
* Thermistor (NTC) unknown value
* Diodes (1N400X)
* Microphone (Electret)
